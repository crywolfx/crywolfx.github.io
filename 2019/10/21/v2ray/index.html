v2ray安装

v2ray新手搭建使用教程

安装curl和vim

  1. RedHat系列：Redhat、Centos、Fedora等 yum install curl vim -y

  2. Debian系列：Debian、Ubuntu等 apt-get install curl vim -y

v2ray脚本安装

<(curl -L -s https://install.direct/go.sh)

v2ray命令执行

  1. 启动：systemctl start v2ray
  2. 检测：systemctl status v2ray
  3. 失败尝试：service v2ray start service v2ray status

关闭防火墙

如果有重要资料请开启对应端口

sudo apt update
sudo apt install nginx

server{
    listen 80;
    server_name example.com;
    return 301 https://$server_name$request_uri; #http重定向至https
}

server {
    listen 443 ssl http2 default_server;
    listen [::]:443 ssl http2 default_server;

    server_name example.com www.example.com;

    ssl_certificate /etc/nginx/ssl/example.com.crt;  #ssl证书
    ssl_certificate_key /etc/nginx/ssl/example.com.key; #ssl证书

    root /var/www/html;  #根目录
    index index.html index.htm index.nginx-debian.html index.php;

    location /ray {  #客户端配置需要和此目录保持一致
        proxy_redirect off;
        proxy_pass http://127.0.0.1:4443;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection "upgrade";
        proxy_set_header Host $http_host;
    }
}
