安装curl和vim

  1. RedHat系列：Redhat、Centos、Fedora等 yum install curl vim -y

  2. Debian系列：Debian、Ubuntu等 apt-get install curl vim -y

v2ray脚本安装

v2ray命令执行

  1. 启动：systemctl start v2ray
  2. 检测：systemctl status v2ray
  3. 失败尝试：service v2ray start service v2ray status

nginx安装

debian9

安装nginx

nginx命令

  • 启动: systemctl start nginx
  • 停止: systemctl stop nginx
  • 重启: systemctl restart nginx
  • 热重载: systemctl reload nginx
  • 开机自启动: systemctl enable nginx
  • 开机不自启动: systemctl disable nginx

配置config

vi /etc/nginx/sites-available/example.com
sudo ln -s /etc/nginx/sites-available/example.com /etc/nginx/sites-enabled/

反向代理
server{
    listen 80;
    server_name example.com;
    return 301 https://$server_name$request_uri; #http重定向至https
}

server {
    listen 443 ssl http2 default_server;
    listen [::]:443 ssl http2 default_server;

    server_name example.com www.example.com;

    ssl_certificate /etc/nginx/ssl/example.com.crt;  #ssl证书
    ssl_certificate_key /etc/nginx/ssl/example.com.key; #ssl证书

    root /var/www/html;  #根目录
    index index.html index.htm index.nginx-debian.html index.php;

    location /ray {  #客户端配置需要和此目录保持一致
        proxy_redirect off;
        proxy_pass http://127.0.0.1:4443;   #端口需要和v2ray配置的端口一致
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection "upgrade";
        proxy_set_header Host $http_host;
    }
}
